> Teeny Node Module Generator is a command line tool that can scaffold out a brand-new node module with an optional CLI entry-point.

## Installation

```sh
> npm install --global yo
> npm install --global generator-teeny-nm
```

## Usage

```sh
> yo teeny-nm
```
